# Amazon Laptop Data Scraping

A Python web scraping project to extract laptop data from Amazon.ae using Selenium and BeautifulSoup.

## Features
Collected laptop listing data from Amazon.ae by browsing multiple pages using Selenium

Managed dynamic web content and page navigation on a JavaScript-based website

Saved web pages locally so data could be extracted later without reloading the site

Extracted product details from saved pages using BeautifulSoup

Organized the collected data into files for further analysis and reporting

## Technologies
- Python
- Selenium WebDriver
- BeautifulSoup4
- Pandas
